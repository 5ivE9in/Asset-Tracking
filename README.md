# Asset-Tracking
Generates qr code and deciphers qr code through camera. Uses OpenCV library to read video frames, to detect QR codes and decode the qr data.


* **Language:** Python 3
* **Libraries:** `opencv-python` (cv2), `pyzbar`, `qrcode`, `Pillow`, `tkinter`

## How to Run Locally
1. Clone this repository to your local machine.
2. Install the required dependencies by running:
   `pip install opencv-python pyzbar qrcode[pil]`
3. Launch the dashboard by running:
   `python main.py`
